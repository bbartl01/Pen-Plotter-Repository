Detailed Description

File that creates a live animation of shape being drawn.


This file receives data over UART to create a live animation of the Pen Plotter drawing the shape that it is currently drawing.

def LivePlotter.Plot ( )

A function that creates the live animation of the Pen Plotter drawing.


This function receives data from UART and uses it to update a plot, thus creating a real-time anmation of the pen plotter drawing the shape.
